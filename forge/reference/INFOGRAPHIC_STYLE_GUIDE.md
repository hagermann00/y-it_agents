# Y-IT Infographic Style Guide

## Style Definition

| Attribute | Specification |
|-----------|---------------|
| Background | **White or light cream** |
| Colors | Flat, muted tones (Gold #D4A84B, Cyan #00D4D4, Gray #444) |
| Lines | Clean vectors, solid colors, NO glow |
| Effects | **ZERO** neon, **ZERO** 3D render, **ZERO** circuit textures |
| Typography | Clean sans-serif, black text |
| Feel | **Editorial cartoon data viz** - newspaper style |

---

## Prompt Suffix

Add this to ALL infographic prompts:

```
Clean editorial infographic style. White background. Flat colors, no glow, no neon, no 3D effects. Simple vector art like a newspaper data visualization. Match PosiBot cartoon aesthetic.
```

---

## Approved Concepts

1. Iceberg Metaphor (visible vs hidden costs)
2. Leaky Pipe Funnel (conversion dropoff)
3. Guru Promise vs Reality (bar chart comparison)
4. Survival Rate Gauge (radial gauge)
5. Capital Burn Rate (line chart)
6. Data Dashboard (metrics layout)
