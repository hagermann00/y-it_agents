## USAGE NOTES

### For NotebookLM / Y-It-itude Writer
- Select ONE epigraph and ONE tagline per chapter
- Match tone to chapter mood (mockumentary CH1-2, brutal CH3-5, sincere CH6-8)
- Customize topic-specific references

### For Claude Polish
- Refine wordplay if needed
- Ensure epigraph source attribution is funny but not mean
- Verify tagline matches chapter content

### Image Placeholders
- Replace bracketed image names with actual filenames after image generation
- Maintain naming convention: [type]-[chapter]-[descriptor].png
